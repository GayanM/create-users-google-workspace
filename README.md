# create-users-google-workspace
You can provide list of users in a csv file and add those users to google workspace via admin API with OAuth authentication

## Python Libraries that need to be installed
pip3 install google-auth google-auth-oauthlib google-api-python-client

## Google Workspace URL
https://admin.google.com/

## Google Cloud URL for OAuth App
https://console.cloud.google.com/

## Additional instructions 
You need to add your client_secret.json into project root which should be downloaded from your OAuth application in Google Cloud platform
